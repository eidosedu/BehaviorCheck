# BehaviorCheck
Ferramenta digital de gestão comportamental para tablets baseada na estratégia Check-In/Check-Out (CICO) e em ABA. Otimiza o monitoramento de contingências em sala de aula com registros em um toque, descrições técnicas rigorosas e exportação instantânea de relatórios via WhatsApp para feedback direto aos pais. Foco em precisão e agilidade
